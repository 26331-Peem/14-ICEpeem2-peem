# Fitness & Calorie tracker
(โปรเเกรมบันทึกการออกกำลังกายคำนวณเเคลอรี่ที่ได้กับคำนวณเเคลอรี่ที่เสียไป)

# ที่มาและความสำคัญ (Introduction)
- ผมสร้างโปรเเกรมนี้ขึ้นมาเนื่องจากคนส่วนใหญ่ที่เพิ่งเริ่มออกกำลังมักจะคำนวณเเคลอรี่หรือออกกำลังกายได้ไม่ถูกต้อง ผมเลยทำโปรเเกรมนี้ขึ้นมาเพื่อทำให้ผู้คนออกกำลังกายเเละคำนวณเเคลอรี่ได้ถูกต้อง

# ความสามารถของโปรแกรม (Features)
- คำนวณเเคลอรี่ที่ได้กับคำนวณเเคลอรี่ที่เสียไป

# เครื่องมือที่ใช้ (Tech Stack)
